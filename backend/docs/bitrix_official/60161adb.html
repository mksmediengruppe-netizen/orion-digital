CIBlockType

CIBlockType - класс для работы с типами информационных блоков.

Методы класса

МетодОписаниеС версии
GetList Возвращает список типов по фильтру.3.1.3
GetByID Возвращает тип по идентификатору.3.1.3
GetByIDLang Возвращает настройки типа для заданного языка.3.1.3
Add Добавляет новый тип информационных блоков.3.1.3
Update Изменяет тип.3.1.3
Delete Удаляет тип информационных блоков.3.1.3
